# Fashion MNIST CNN

Jupyter notebook: `fashion_mnist_cnn.ipynb`

Description:
- Convolutional Neural Network trained on the Fashion MNIST dataset.

How to run:
1. Open the notebook in Jupyter, JupyterLab, or Google Colab.
2. Install dependencies (example):

```
pip install tensorflow numpy matplotlib
```

Notes:
- This folder is prepared to become a separate repository. Use the provided `scripts/package_repos.py` helper to copy the notebook into this folder, initialize a git repo, and optionally push to GitHub.
